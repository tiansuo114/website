### Synopsis

Shows information about the images required for Karmada deployment.

 This command lists all the images needed to deploy and run Karmada, including images for the API server, controller manager, scheduler, and other components.

### Examples

```
  # List all required images from the specified private image registry
  karmadactl config image list
  
  # Use a specific registry for Karmada images
  karmadactl config image list --private-image-registry registry.cn-hangzhou.aliyuncs.com/google_containers
```
