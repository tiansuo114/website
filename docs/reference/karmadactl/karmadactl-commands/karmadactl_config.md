### Synopsis

Manage Karmada configurations.

 The 'config' command allows you to manage the configurations required for deploying and running Karmada. This includes listing, setting, and modifying various configurations such as image registries, versions, and other settings.

### Examples

```
  # List all images required for deploying Karmada
  karmadactl config image list
  
  # Use a specific registry for Karmada images
  karmadactl config image list --private-image-registry registry.cn-hangzhou.aliyuncs.com/google_containers
```
